Sobre o Scanner IA Master

Ferramenta com IA para diagnóstico de falhas OBD2. Basta selecionar veículo, modelo, inserir o código DTC e obter um laudo técnico.

📘 Como utilizar:

  • 1. Configure sua API Key Gemini (salva localmente).
  • 2. Escolha Carro, Caminhão ou Moto.
  • 3. Selecione Marca e Modelo via FIPE.
  • 4. Digite o código de erro e clique em ANALISAR.
  • 5. Use o chat para perguntas adicionais.
